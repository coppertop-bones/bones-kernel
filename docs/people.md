Software development is more than coding and a team of skills is needed.

We have target audiences:

kernel user \
python user \
bones user \
kernel keeper \
proxy user acts as a user forming direction - who ask are we doing / have we done the right thing? \
authors explain and teach


The overall vision is to give the world a language / system that non-career programmers can use to access data based 
decision-making tools including wrangling, statistics and algos and thus help them form opinions on courses of action 
they would otherwise struggle to form.
